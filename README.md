# Knative (knative)
Knative is a CNCF graduated platform providing serverless capabilities on Kubernetes. It includes Serving for auto-scaling workloads with scale-to-zero, and Eventing for building event-driven architectures with CloudEvents support.

## APIs

### Knative Serving API
Knative Serving extends the Kubernetes API with custom resources for deploying serverless workloads including automatic scaling, scale-to-zero, traffic splitting, and progressive rollouts.

**Human URL:** [https://knative.dev/docs/serving/](https://knative.dev/docs/serving/)

#### Tags:
 - Serverless, Auto-Scaling, Serving

#### Properties
- [Documentation](https://knative.dev/docs/serving/)

### Knative Eventing API
Knative Eventing provides Kubernetes custom resources for building event-driven architectures including Broker/Trigger for routing and Channel/Subscription for pub/sub messaging.

**Human URL:** [https://knative.dev/docs/eventing/](https://knative.dev/docs/eventing/)
